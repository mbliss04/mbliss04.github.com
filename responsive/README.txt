1. Nodice.css is utilized as the stylesheet when the screen
	size is greater than 481px. This CSS file will then
	be used when on a desktop computer or a tablet.

2. Dice.css is put into effect when the screen size is less
	than 481px. This means that it not only shows up on
	mobile devices, but also when the screen on a desktop
	is shrunk down less than 481 px.
